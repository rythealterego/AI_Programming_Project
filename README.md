## Project Description
This project demonstrates essential Python programming concepts required in AI development including functions, file handling, user input, data manipulation, and modular programming.

## Python Libraries Used
- os — file and system operations  
- random — generating randomized values  
- json — data storage and exchange  
- datetime — timestamps and date handling  


## How to Run the Code

1. Download or clone this repository to your computer.
2. Open the folder in Visual Studio Code.
3. Make sure Python 3 is installed on your computer.
4. Open a terminal in VS Code.
5. Navigate into the chatbot folder:  cd chatbot
6. Run the program using:  python main.py
7. Type your questions in the terminal and the chatbot will respond.
